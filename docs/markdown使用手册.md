# markdown使用手册

# 标题
# 一级标题
## 二级标题
### 三级标题
注意符合“#”与标题之间有空格

# 粗体
**这是粗体**
__这是粗体__
 
# 斜体
*这是斜体*
_这是斜体_
 
# 换行
“Enter”键最多换一行，换多行用<br/>
 
# 分割线
“Enter”键空一行
_ _ _ 或者 ***
 
# 引用
>文字(可以嵌套)
 
# 无序列表
*/+/- 文字
 
# 有序列表
1./2./3. 文字
 
# 行内代码块
`代码块`
`` ` ``
 
# 段落代码块
```python/sh/C++
code
```
注意符号“`”是英文输入条件下触发的Esc下面的按键
